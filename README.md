# dingdian

----

	<ul>
		<li>selenium</li>
		<li>PhantomJS</li>
		<li>ChromeDriver
			<a href="https://sites.google.com/a/chromium.org/chromedriver/downloads">下载地址</a>
		</li>
		<li>python3.x</li>
		<li>lxml</li>
	</ul>
